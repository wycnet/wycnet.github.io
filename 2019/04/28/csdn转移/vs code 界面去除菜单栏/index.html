vs code界面的设置

vscode在mac和ubuntu下的界面颜色纯黑，比较好看。但是win10下上部菜单大片的白色，丑！！！！

打开设置，从设置里找到窗口

找到倒数第二个选项，点击编辑按钮

选择custom选项，重启后界面就变成了如下图

或者，你可能会觉得菜单在窗口上也很丑，有没有方法隐藏它？
当然有方法，同样在设置中的窗口目录中”window.menuBarVisibility”: “default”, 设置为toggle，如下图所示：

完成后，会发现菜单栏被隐藏，按住alt键，菜单栏会出现。
