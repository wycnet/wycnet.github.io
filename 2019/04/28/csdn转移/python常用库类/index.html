threading 模块

threading.Thread

threadong 中的Thread类，可以使用它来创建多线程。
具体使用方法是创建一个threading.Thread对象。

threading.Lock

使用lock.acquire() 和lock.release()包围的代码将进行同步操作，即此代码块执行完毕后，才能进行线程切换。

threading.join

join类是threading中用于阻塞当前主线程的类，其作用是阻塞当前所用线程，直到被调用的线程执行完毕或者超时。
